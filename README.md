# discord-board-games

## Minimax and DeepNN Tictactoe Bot

prefix tictactoe [bot | solo | random | nn]: Starts a tictactoe game

Tictactoe bot is basic minimax.

Tictactoe nn uses a self trained deep nn.

## DeepNN and NNUE Minimax Connect 4 Bot

prefix connect4 [bot | solo | random | nn]: Starts a connect4 game

Connect4 nn uses a self trained (currently not that trained) deep nn.

Connect4 bot uses minmax with nnue and basic alpha beta pruning with basic iterative deepening.

## Roll

prefix roll [number]: Rolls a number between 1 and [number] provided (defaults to 100)

# discord-board-games

prefix roll [number]: Rolls a number between 1 and [number] provided (defaults to 100)

prefix tictactoe: Starts a game of tic tac toe against another user

prefix tictactoe bot: Starts a game of tictactoe against the bot

prefix tictactoe random: Starts a game of tictactoe against the random bot

prefix tictactoe solo: Starts a game of tictactoe against yourself
